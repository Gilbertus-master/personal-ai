# Gilbertus Desktop Icons

Generate icons from the Gilbertus logo using `cargo tauri icon <source.png>`.

## Required files

| File | Size | Platform |
|------|------|----------|
| `icon.ico` | multi-size | Windows |
| `icon.icns` | multi-size | macOS |
| `icon.png` | 512x512 | Linux |
| `32x32.png` | 32x32 | All |
| `128x128.png` | 128x128 | All |
| `128x128@2x.png` | 256x256 | macOS Retina |
| `Square30x30Logo.png` | 30x30 | Windows |
| `Square44x44Logo.png` | 44x44 | Windows |
| `Square71x71Logo.png` | 71x71 | Windows |
| `Square89x89Logo.png` | 89x89 | Windows |
| `Square107x107Logo.png` | 107x107 | Windows |
| `Square142x142Logo.png` | 142x142 | Windows |
| `Square150x150Logo.png` | 150x150 | Windows |
| `Square284x284Logo.png` | 284x284 | Windows |
| `Square310x310Logo.png` | 310x310 | Windows |
| `StoreLogo.png` | 50x50 | Windows |
